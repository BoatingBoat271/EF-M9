# Portfolio Web

Portfolio profesional con secciones de habilidades, proyectos y contacto.

## Requisitos

- Navegador web moderno
- No requiere Node.js ni build

## Instalacion y uso local

```bash
# 1) Entrar a la carpeta
cd portfolio
```

Opciones para ejecutarlo:

```bash
# Opcion A: abrir index.html directamente
# Opcion B: servidor local (recomendado)
npx --yes serve . -l 5501
```

Abre `http://localhost:5501`.

## Archivos principales

- `index.html`
- `styles.css`
- `img/`
- `assets/img/`
