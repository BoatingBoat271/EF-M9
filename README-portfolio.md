## Descripción

## Instalación y uso

No requiere instalación ni dependencias. Solo abre el archivo en tu navegador:

```bash
# Opción 1: Abre directamente desde el explorador de archivos
Doble clic en index.html

# Opción 2: Usando Live Server (VS Code)
Clic derecho en index.html → "Open with Live Server"
```

> **Nota:** Antes de publicar, reemplaza todos los textos `TU_LINK_*` con tus URLs reales.

---

## Personalización

1. Abre `index.html`
2. Reemplaza los textos marcados con `TU_LINK_*` con tus enlaces reales:
   - `TU_LINK_DEMO_RECETAS` → URL del deploy del Buscador de Recetas
   - `TU_LINK_GITHUB_RECETAS` → URL del repositorio del Buscador de Recetas
   - `TU_LINK_DEMO_KANBAN` → URL del deploy de Kanban Pro
   - `TU_LINK_GITHUB_KANBAN` → URL del repositorio de Kanban Pro
   - `TU_EMAIL@gmail.com` → Tu email real
   - `TU_USUARIO` → Tu nombre de usuario en LinkedIn y GitHub
